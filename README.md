  <br> <br>

installation
```js
npm i ping-server
```

<br>
  
calling variable
```js
const ping = require("ping-server");
```

<br>

how to use?
```js
ping.on("YOUR LINK");
```

<br>

example:
```js
ping.on("https://web.leiamnash.repl.co");
```
